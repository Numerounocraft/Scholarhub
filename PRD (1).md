# 🌍 Scholarship Aggregator Web App - Product Requirements Document (PRD)

## 📌 Overview
A free web application that aggregates global scholarship opportunities and notifies users based on their preferences.

---

## 🎯 Goals
- Provide up-to-date scholarship listings
- Enable personalized alerts
- Operate entirely on free-tier tools

---

## 👤 Target Users
- Students (Undergrad, Masters, PhD)
- International applicants
- Scholarship seekers

---

## 🧩 Core Features (MVP)

### 1. Scholarship Feed
- Title
- Country
- Degree level
- Deadline
- Link
- Description

### 2. Filtering & Search
- Country
- Degree
- Field
- Deadline

### 3. User Preferences
- Save interests for personalization

### 4. Email Notifications
- Alerts based on preferences

### 5. Admin Dashboard
- Add/edit scholarships

---

## 🏗️ Tech Stack (Free)

- Frontend: Next.js (Vercel)
- Backend: API Routes (Serverless)
- Database: Supabase / Firebase
- Auth: Supabase Auth
- Email: Resend / SendGrid
- Cron: GitHub Actions

---

## 🔄 User Flow

1. User signs up
2. Sets preferences
3. Browses scholarships
4. Receives alerts

---

## 📊 Success Metrics
- User growth
- Email open rate
- Click-through rate

---

## 🚀 Timeline

Week 1: Setup + UI  
Week 2: Auth + Filtering  
Week 3: Notifications  
Week 4: Launch  

---

## 🔥 Key Principle
Keep it simple, free, and scalable.
